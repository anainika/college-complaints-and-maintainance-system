The College Complaint and Maintenance System is a web-based application designed to streamline the process of reporting, managing, and resolving complaints within a college campus. It replaces traditional paper-based complaint registers with a centralized, digital system that allows students and administrative staff to communicate effectively regarding campus issues.
Features
Student Portal
Register/Login using Roll Number and Email.
Submit complaints with:
Category (Hostel, Classroom, Lab, etc.)
Description
Location / Block
Optional photo upload
Track complaint status: Pending → In Progress → Resolved.
View assigned staff and admin remarks.
Admin Portal
Secure login with username and password.
View all student complaints in a dashboard.
Filter complaints by Category, Block, and Status.
Assign complaints to staff members.
Update complaint status and add remarks.
View uploaded images for verification.
Technologies Used
Frontend: HTML, CSS, JavaScript
Storage: LocalStorage (for storing student info and complaints)
No backend or database required (fully client-side)
